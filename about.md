# Layers Offset

I know some players (me, for example) who usually allocate 5-10 layers per any level's part aspect - block design, air decorations, background, etc.,
and I know sometimes they run into issue that they either need more layers per detail or less layers, resulting into long manual layer reordering.
This mod is designed to solve this and make empty layers not a big problem.

## Description

This mod adds a new button in edit tab menu in the editor, where you can specify layers to move and the value by which they should be moved.
It also supports changing the editor layer 2 property as well.

## How to use

1. Place some objects on different layers.
2. Go to Edit tab in bottom editor menu.
3. Find a new reoffset button added by the mod and click on it.
4. Set the needed layers as regular values or ranges, you can divide them by a comma.
5. Set the reoffset value (can be a negative number).
6. Hit the editor layer 2 checkbox if you wish to change the editor layer 2, not the regular layer.
7. Confirm the changes and see the mod do the job! (you can also undo the changes if you mess up)

## Compatibility

I have many editor mods installed and I never ran into any issues with them. If there is any incompatibility, let me know by creating an issue on [GitHub repository](https://github.com/zL4mpY/clone_and_move_mod) or writing me in Discord DMs (eni.meow).

The mod has a keybind to quickly open the menu if you need. It can be configured in the settings or the keybinds menu.
